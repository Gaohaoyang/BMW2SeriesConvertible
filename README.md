# BMW 2 series convertible official used cars in China

本仓库是用于搜索宝马官网的二手车信息，当时我在购买2系敞篷时使用。

在线地址 https://gaohaoyang.github.io/BMW2SeriesConvertible/

目前我已买到心仪的车辆，暂时不会更新此链接数据，若你想使用此方式快速寻找车辆，可继续往下看。

## Usage

获取数据命令

```
yarn u
```

本地启动服务

```
yarn start
```

即可访问

http://localhost:3000/
